## ğŸ”¬ How It Works

1. **Upload** an image (JPG/PNG) of a digit.
2. **Preprocessing Pipeline:**
   - Convert to grayscale
   - Invert colors if needed (white digit on black background)
   - Crop to the digit bounding box
   - Resize to **8Ã—8 pixels** with smooth gradients
   - Scale pixel values to **0â€“16** range (matching training format)
3. **Predict** using the pre-trained SVM model.
4. **Display** the predicted digit and the processed 8Ã—8 input.
